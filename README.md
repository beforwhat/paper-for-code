# FedFairADP-ALA
联邦学习公平性优化框架：融合SA贡献度（Shapley值）与自适应裁剪差分隐私（ADP），新增ALA特征提取模块，提升联邦学习的公平性、隐私性与效率。

## 1. 环境要求
| 组件         | 版本/要求                          | 说明                                  |
|--------------|------------------------------------|---------------------------------------|
| 操作系统     | Linux Ubuntu 20.04/22.04（推荐）| 对GPU驱动/CUDA支持最好，稳定无坑      |
| Python       | 3.9.x                              | 核心适配版本，兼容所有依赖            |
| GPU          | NVIDIA RTX 3090/A10/A100（显存≥24G） | 并行训练/Shapley值计算加速必备        |
| CUDA         | 11.8                               | 匹配PyTorch 2.1.0，最大化GPU利用率    |
| pip          | ≥21.0                              | 依赖安装工具                          |
